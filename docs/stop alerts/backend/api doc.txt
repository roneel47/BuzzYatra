base url of open street map API-
https://api.openstreetmap.org/

the key components present in it are 
1. Map Zoom
2. latitude
3. Longitude
4. type of layer

https://www.openstreetmap.org/#map=20/12.9987861/77.5842060&layers=T

consider this as an example

#map represents the zoom level
12.9987861 represents latitude
77.5842060 represents longitude
&layer=T represents the type of layer and in this case it is transport

	// OpenStreetMap API Example
	const response = await fetch('http://wiki.openstreetmap.org/wiki/API', {
